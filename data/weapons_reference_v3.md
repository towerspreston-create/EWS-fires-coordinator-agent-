### 8.2 Salvo Calculation Method
To achieve desired Pk with multiple rounds:

**Formula:**
```
n = log(1 - Pk_desired) / log(1 - Pk_single)

Where:
  n = number of rounds required (round up)
  Pk_desired = probability of kill you want to achieve (e.g., 0.90 for 90%)
  Pk_single = single-round Pk from table above
```
